== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accordion-guttenberg-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Open Post or Page in WordPress admin
4. Select Accordion Block in gutenberg blocks
5. Paste title and contect
